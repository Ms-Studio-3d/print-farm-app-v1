Print Farm App
تطبيق سطح مكتب مبني بـ Electron لإدارة الطباعة ثلاثية الأبعاد، وحساب التكلفة، ومتابعة المخزون، وتسجيل المبيعات في واجهة عربية بسيطة وسريعة.

المميزات الحالية
حساب تكلفة الطباعة بناءً على:
استهلاك الخامة
إهلاك الماكينة
الشغل اليدوي
التغليف وهامش الفشل
هامش الربح
إدارة مخزون الخامات
إضافة بكرات جديدة
تسجيل المبيعات
تعديل وحذف الأوردرات
تقارير وإحصائيات للمبيعات
تصدير واستيراد نسخة احتياطية بصيغة JSON
بناء نسخة Windows Installer باستخدام GitHub Actions
